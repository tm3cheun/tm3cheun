Real-time Kernel
Developed a small real-time kernel for ARM Cortex-M3 Processor. Supports task creation, multithreading, I/O (GPIO and peripherals), and system console on KEIL MCB1700 evaluation board. Manages memory using half-fit allocation algorithm with fixed memory blocks.
